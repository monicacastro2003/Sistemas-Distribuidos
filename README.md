**Professor:** Rafael Braga  
**Linguagem:** C++  
**Tema:** Comunicação entre processos  

---

## Descrição do Projeto  

Este projeto implementa um sistema distribuído de **Gestão de Supermercado**, onde o **cliente** e o **servidor** se comunicam através de **sockets TCP/UDP**.  
O objetivo é simular o gerenciamento básico de um supermercado em **C++**, utilizando conceitos de **serialização**, **empacotamento de dados** e **comunicação entre processos**.
